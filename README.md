# Forced-Alignment-using-Montreal-Forced-Aligner-MFA-
Set up and execute a complete forced alignment pipeline using the Montreal Forced Aligner (MFA) tool. And to understand how automatic alignment works between speech audio and phonetic transcription.

**For this to work:**
1. Clone the repository
2. Run the below command in the terminal
   >>chmod +x run_alignment.sh && ./run_alignment.sh
3. Get the textgrid files in the output_grids folder
4. Get the custom dictionary saved as custom_dict.txt to handle the OOV words.
